# My Dotfiles

Personal configuration files for Linux development environment.

## Features

- Zsh with oh-my-zsh and powerlevel10k
- Vim with LSP support
- Python and C/C++
- FZF fuzzy finding

## Installation

```bash
git clone ssh@github.com:Madfury0/.dotfiles.git ~/.dotfiles
cd ~/.dotfiles
./scripts/setup.sh
